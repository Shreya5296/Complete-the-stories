Create virtual env using following commands to run this project:

#### Windows:
1. python3.8 -m venv venv
2. venv\Scripts\activate.bat
3. pip install flask
4. python app.py

#### MacOS/Linux:
1. python3.8 -m venv venv
2. venv/Scripts/activate
3. pip install flask
4. python app.py
